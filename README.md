# 🧮 Calculator Website

This repository contains a **simple calculator website** built using **HTML, CSS, and JavaScript**. The calculator allows users to perform basic arithmetic operations directly in the browser through an easy-to-use interface.

## 📁 Repository Files

* **index.html** – Contains the structure of the calculator interface
* **style.css** – Handles the styling and layout of the calculator
* **script.js** – Implements the calculation logic and button functionality

## ✨ Features

* Perform basic arithmetic operations:

  * Addition (+)
  * Subtraction (−)
  * Multiplication (×)
  * Division (÷)
* Simple and clean user interface
* Interactive buttons using JavaScript
* Instant display of results

## 🛠️ Technologies Used

* **HTML5** for the webpage structure
* **CSS3** for styling the calculator
* **JavaScript** for handling calculations and user input

## 💻 How to Run the Project

1. Clone the repository

git clone https://github.com/sarojbaraily11-lang/CODSOFT_TASK2.git

2. Open the project folder.

3. Open **index.html** in your web browser to use the calculator.

## 🎯 Purpose

This project was created to practice **basic front-end web development concepts** such as structuring a webpage, styling with CSS, and adding interactivity using JavaScript.
